1.  首先关闭 docker
2.  关闭所有发行版：`wsl --shutdown`
3.  将 docker-desktop-data 导出到 D:\\u2004.tar（注意，原有的 docker images 不会一起导出） `wsl --export docker-desktop-data D:\u2004.tar`
4.  注销 docker-desktop-data： `wsl --unregister docker-desktop-data`
5.  重新导入 docker-desktop-data 到要存放的文件夹：D:\\Program Files\\wsl-docker-data： `wsl --import docker-desktop-data "D:\Program Files\wsl-docker-data" D:\docker-desktop-data.tar --version 2`

重启启动 docker

好了，接下来就是改一下 docker 的 registry-mirrors ，指向自己喜欢的加速器，愉快的拉镜像去吧，还不用担心“C 盘空间不足焦虑症”发作。。。
